# Shannon Lo — SJSU A.S. Vice President Campaign Website

A React-based campaign website for Shannon Yu Hsin Lo, running for Associated Students Vice President at San Jose State University (2026–2027).

## Tech Stack

- React (Create React App)
- Three.js — liquid ether fluid simulation on hero section
- OGL — grainient animated gradient on letter section
- CSS custom properties + Google Fonts (Playfair Display, DM Sans)

## Getting Started
```bash
npm install
npm start
```

## Build
```bash
npm run build
```

## Project Structure
```
src/
  assets/          # Images and logo
  components/
    Nav.jsx         # Fixed nav with bubble menu
    Hello.jsx       # Hero section with liquid ether effect
    About.jsx       # Headshot + bio
    Volunteers.jsx  # Campaign team cards
    Endorsements.jsx
    Involvement.jsx # Circular gallery of orgs
    CircularGallery.jsx
    Letter.jsx      # Open letter with grainient background
    Priorities.jsx  # Stacked card priorities
    Hobbies.jsx     # Horizontal scroll strip
    LetsTalk.jsx    # Contact/footer
    LiquidEther.jsx # Three.js fluid simulation
    Grainent.jsx    # OGL animated gradient
```
